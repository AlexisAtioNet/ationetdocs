## Measurement Unit Codes

Unit codes are used to specify the volume or mass unit of the product. However, no unit conversions will be made in the host and ppu's will always be in the reported unit.

| Value | Description |
| ----- | ----------- |
| usgal | USA Gallon  |
| ukgal | UK Gallon   |
| l     | Litre       |
| m3    | Cubic metre |
| kg    | Kilogram    |
